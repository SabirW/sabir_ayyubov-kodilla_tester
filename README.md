# Kodilla Test Automation Suite

A comprehensive test automation project developed as part of the 
Kodilla QA & Test Automation Bootcamp.

## 🛠️ Tech Stack
- Java
- JUnit
- REST Assured
- Gradle
- Git

## 📁 Project Structure
- `kodilla-basic-tests` — fundamental test case design and JUnit basics
- `kodilla-advanced-tests` — advanced testing scenarios and patterns
- `kodilla-rest-api` — REST API validation with REST Assured
- `kodilla-collections` — Java collections testing
- `kodilla-collections-advanced` — advanced collections scenarios
- `kodilla-stream` — Java Stream API tests
- `kodilla-spring-basic` — basic Spring framework integration

## 🚀 How to Run
1. Clone the repository
   git clone https://github.com/SabirW/sabir_ayyubov-kodilla_tester.git
2. Open in IntelliJ IDEA
3. Run tests via Gradle:
   ./gradlew test
